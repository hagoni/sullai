기본배치 규정

Basic Positioning

심볼마크는 그리드 규정을 준수하여
기본 형태의 오류가 발생하지 않도록 제작하여야 하며,
시그니처 상하조합의 높이(K)를 기준으로 하여
주변에 어떠한 요소도 침범하면 안되는
최소의 공간규정을 준수하여야 합니다.

전용색상

Color System

  • Main Color
    강강술래 심볼마크를 구성하는 색이며
    개별적인 색의 활용보다는
    세 색의 배색을 통하여 강강술래의 칼라
    아이덴티티를 표현할 수 있습니다.

  • Sub Color
    시그니처나 엠블럼 등에 추가적으로
    사용할 수 있는 포인트 칼라로
    basic color의 이미지를 손상시키지 않는
    범위에서 사용 할 수 있습니다.
